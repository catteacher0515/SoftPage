# Cover MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a standalone 9:16 cover-making workflow that auto-fills title from Markdown, accepts one manually uploaded hero image, previews the fixed five-zone cover layout, and exports one PNG separately from body pages.

**Architecture:** Add a dedicated cover draft state alongside the existing body pagination state, then introduce a lightweight editor mode switch between body layout and cover creation. Cover preview and export use their own rendering/export path, while still sharing Markdown title extraction from the current source document.

**Tech Stack:** Next.js, React, TypeScript, html2canvas, Jest, Testing Library

---

Create `components/export/export-cover.ts`:

```ts
import html2canvas from 'html2canvas'

export async function exportCoverAsPng(element: HTMLElement) {
  const canvas = await html2canvas(element, {
    backgroundColor: null,
    scale: 2,
    useCORS: true,
  })

  const url = canvas.toDataURL('image/png')
  const link = document.createElement('a')
  link.href = url
  link.download = 'softpage-cover.png'
  link.click()
}
```
